# Talkamrater

A simple web app built with Om Next and Bootstrap.

# Develop
To start development with figwheel:
```
rlwrap lein run -m clojure.main script/figwheel.clj
```

# Deploy
To build and push artifacts to gh-pages:
```
deploy.sh
```
